Medium post statistics Starts a background job that uses the configured Medium.com integration to open a signed-in browser session and scrape medium.com/me/stats. Results are stored as medium_stats_report.json on the job. Under Integrations, save encrypted Google credentials for Medium and/or capture a browser session; the stats job will open medium.com/me/stats using the stored session (or sign in automatically if credentials are saved and the session file is missing).
