Medium post statistics Starts a background job that uses the configured Medium.com integration to open a signed-in browser session and scrape medium.com/me/stats. Results are stored as medium_stats_report.json on the job. Enable the integration and import a Playwright storage_state under Integrations. The session file location is controlled by backend environment settings and is not configurable in the UI.
