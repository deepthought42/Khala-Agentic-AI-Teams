# Inclusive Experience Agency Service Catalog

## Core Packages

### 1. Baseline Accessibility Audit
- **Use when:** A product team needs rapid visibility into major WCAG 2.2 risks.
- **Coverage:** Critical journeys, high-traffic templates, key components.
- **Output:** Prioritized finding backlog with evidence and remediation guidance.

### 2. Comprehensive Accessibility Assessment
- **Use when:** Teams need broad WCAG + Section 508 confidence across web and mobile.
- **Coverage:** Web journeys, mobile experiences, AT validation, standards mapping.
- **Output:** Full report pack, coverage matrix, executive summary, and roadmap.

### 3. Retest and Verification Sprint
- **Use when:** Teams have implemented fixes and need closure evidence.
- **Coverage:** Targeted findings and regression checks.
- **Output:** Closed findings with verification evidence and remaining risk register.

## Add-On Services

### Accessibility Enablement Program (AET)
- Pattern mining from audit results.
- Team-specific micro-training modules.
- Repeat issue prevention guidance.

### Continuous Regression Monitoring (ARM)
- Baseline creation for selected surfaces.
- Scheduled drift checks with severity-aware alerts.
- Trend summaries for governance stakeholders.

### Design System Hardening (ADSE)
- Component accessibility contract templates.
- Definition of done criteria for reusable components.
- Component-level regression check recommendations.
