# Integration and Release Team

Post-execution agents that integrate or package the product after backend and frontend workers complete.

| Agent | Role |
|-------|------|
| **Integration Agent** | Validates backend–frontend API contract alignment after workers complete |
| **DevOps Expert** | CI/CD, containerization; triggered by Tech Lead for backend/frontend repos |
| **Documentation Agent** | README and project docs; triggered per task and final pass |

These agents run after implementation (and quality gates). Tech Lead triggers DevOps and Documentation; Integration runs once in the orchestrator.
